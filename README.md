# Bibek sales dashboard (data-analysis-dashboard, interactive dashboard creation using powerBI, MS Excel)
📊 *Sales Dashboard (Power BI)* An interactive sales analytics dashboard built using Power BI and Excel to track key business metrics such as Total Sales, Profit, Quantity Sold, and Discount. The dashboard provides visual insights into sales performance across regions, categories, and time periods, helping users make data-driven business decisions.
Data Source: Superstore Sales Dataset – a sample retail dataset widely used for data analysis and business intelligence projects.
             The dashboard uses a sales dataset imported from Excel, containing order details, customer information, product categories, sales, profit, quantity, and discount data for business performance analysis.
Tools & Technologies Used

* Power BI
* Microsoft Excel
* Power Query
* Data Modeling
* DAX (Data Analysis Expressions)

These tools were used for data cleaning, transformation, modeling, KPI creation, and interactive dashboard visualization to generate meaningful business insights from sales data.
Features

✔ Interactive Dashboard
✔ Sales, Profit, Quantity & Discount Analysis
✔ Regional & Category-wise Performance Tracking
✔ Power Query Data Transformation
✔ Data Modeling & DAX Calculations
✔ Dynamic Filters and Visual Insights
